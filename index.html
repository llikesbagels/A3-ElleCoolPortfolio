BEAR CHARACTER SHEET

"Crunk the Bear"

Digital Character Sheet, made using Photoshop, When creating this drawing, I focused a lot on using gradient maps to colour Crunk. It was an interesting learning experience, I had never started colouring in only grayscale before and even though I still don't prefer it, I did enjoy learning how it made me think harder about value shading.

MYLAR ILLUSTRATION

"Maria"

Abstract Mixed Media Artwork made on a 18 x 24 sheet of mylar plastic with graphite pencil, coloured pencils and alcohol-based markers. ​ This project was a real chance to prove that I know what I'm doing in regards to realism and detail. Working with mylar is really mean, it smears like crazy and trying to erase more moves what you're trying to erase than actually erasing it. I loved working with the new material though, it was super smooth and softer graphite really do glide across it like butter
